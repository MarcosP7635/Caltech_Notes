3 Chapter 2

3.1 A Hilbert space is an infinite-dimensional vector space

Every vector in a Hilbert space ℋ maps each real number x to a complex number ψ⁢(x) if the function ψ has a finite norm (we will define norm shortly). The inner product of ψ1⁢(x) and ψ2⁢(x) is ⟨ψ1⁢(x)|ψ2⁢(x)⟩ where we further define

⟨ψ1⁢(x)|ψ2⁢(x)⟩=∫-∞+∞ψ1*⁢(x)⁢ψ2⁢(x)⁢𝑑x (3)

The norm of a function is defined to be the inner product of a function with itself, and by the definition of absolute value in Equation 2

Norm⁢(ψ⁢(x))=∫-∞+∞∣ψ⁢(x)∣2⁢𝑑x (4)

We will now show that

⟨ψ2*⁢(x)|ψ1⁢(x)⟩=⟨ψ1⁢(x)|ψ2⁢(x)⟩* (5)

By definition ψ2⁢(x) and ψ1⁢(x) are both complex functions of a real number, and thus for some f,g,h, and l that are all real functions of the real variable x

ψ2⁢(x)=h+i⁢l,ψ1⁢(x)=f+i⁢g⟨ψ2*⁢(x)|ψ1⁢(x)⟩=∫-∞+∞(h-i⁢l)⁢(f+i⁢g)⁢𝑑x=∫-∞+∞h⁢f+l⁢g+i⁢(h⁢g-f⁢l)⁢d⁢x⟨ψ1⁢(x)|ψ2⁢(x)⟩*=∫-∞+∞(f-i⁢g)⁢(h+i⁢l)⁢𝑑x*=∫-∞+∞h⁢f+l⁢g-i⁢(-h⁢g+f⁢l)⁢d⁢x□ (6)

Two vectors in Hilbert space are orthogonal if their inner product is zero. A set of Hilbert space vectors are orthonormal if each element of the set is orthogonal to all the other elements of the set and each element has a norm of 1. A set of Hilbert space-vectors are complete if every vector in the Hilbert space is a linear combination of the elements of the set. A set of Hilbert space-vectors that is both orthonomal and complete is an orthonormal basis set. We define the delta function

δ⁢(x)=lima→0⁡1|a|⁢π⁢e-(xa)2 (7)

and further define

δi⁢j={1if ⁢i=j0if ⁢i≠j (8)

Thus {ψi⁢(x)} is an orthonormal set iff ⟨ψi⁢(x)|ψj⁢(x)⟩=δi⁢j
Similarly, {ϵi⁢(x)} is an orthonormal basis set iff ⟨ϵi⁢(x)|ϵj⁢(x)⟩=δi⁢j for all positive integers i and j.
A normed vector has a norm of 1 (it is synonymous with being normalized).

3.2 A^ is an operator, which is a function where A^⁢ψ=A^⁢(ψ)

A^ is pronounced A-hat and an operator A^ is hermitian iff

⟨ϕ1|A^⁢ϕ2⟩=⟨A^⁢ϕ1|ϕ2⟩ (9)

Consider if A^ is Hermitian and has an eigenvector ϕ⁢(x) with eigenvalue c. Then we have by the definitions of eigenvector and eigenvalue A^⁢ϕ=c⁢ϕ, and thus we have both (ϕ1,A^⁢ϕ2)=(ϕ1,c⁢ϕ2) and (A^⁢ϕ1,ϕ2)=(c⁢ϕ1,ϕ2). Since A^ is Hermitian we have (A^⁢ϕ1,ϕ2)=(ϕ1,A^⁢ϕ2), which shows (ϕ1,c⁢ϕ2)=(c⁢ϕ1,ϕ2). By our definition of the inner product, the equation (ϕ1,c⁢ϕ2)=(c⁢ϕ1,ϕ2) can only be true if c*=c, which is only true if c is real. Thus, the eigenvalues of a Hermitian operator are always real numbers. The converse is also true :)
For some Hermitian operator A^ the set of its eigenvectors {αn}=α1⁢(x),α2⁢(x),… form an eigenbasis iff {αn} is an orthonormal basis set.

4 Chapter 4

4.1 Postulates of Quantum Mechanics

Consider a particle of mass m that can only move in the x direction and experiences some potential given by V⁢(x)

  1. 1.

    Every physical state corresponds to some normed Hilbert space vector that contains everything that can be known about the system and define the wave function

    ψ⁢(x,t)=ψt⁢(x) (10)

    One must know that this is still only a function of x and that t only serves to label different vectors within the Hilbert space.

  2. 2.

    There is a one-to-one correspondence between every physical observable A and a Hermitian operator A-hat with eigenbasis {αn} such that for every positive integer, we have

    A^⁢αn⁢(x)=An⁢(x)⁢αn⁢(x) (11)

    where the only possible values of the observable A are the eigenvalues An⁢(x)

    If n is a discrete index, then the eigenvalues are discrete and the values of A are quantized :)
    Although the value output wave function can change with time t, the operator corresponding to A is independent of time.

  3. 3.

    The probability that a measurement of the physical observable A will yield an individual eigenvalue

    Probability⁢(Ak)=|⟨αk|ψt⟩|2 (12)

    is where the eigenbasis of A^ is {αi}. This is strongest possible prediction of the measurement.
    Thus if two measurements are done on a system with the same state, even with infinite precision, the two measurements may disagree.

  4. 4.

    Regardless of the state vector before a measurement, immediately after a measurement a state vector will be of the form c⁢αn where |c|=1 and the true value of the measurement was An.

  5. 5.

    Every physical system has a Hermitian operator H^ named the Hamiltonian with an eigenbasis {ηk} where

    H^⁢ηk=Ek⁢ηk (13)

    and the only allowed values of the total energy of the system are {Ek}. Furthermore, as long as the system isn’t disturbed

    i⁢ℏ⁢∂∂⁡t⁢ψ⁢(x,t)=H^⁢ψ⁢(x,t) (14)

    Importantly, Equation 14 is known as the Schrodinger Equation.

  6. 6.

    Position, X, and momentum, P, have the corresponding operators X^=x and P^=-i⁢ℏ⁢dd⁢x

As shown in Section 3.2, if A^ is hermitian then all of its eigenvalues are real. Thus the function f⁢(A) is also a physical observable if f⁢(A) is of the form

f⁢(A)=∑n=0∞cn⁢An,cn∈ℝ⁢∀n (15)

where ℝ refers to the set of all real numbers

4.2 Schrodinger Equations

We will now derive the time independent and time dependent Schrodinger Equations.
Since total energy is the sum of kinetic energy and potential energy we have for Hamiltonian Operator

H^=-12⁢m⁢P^2+V⁢(X^) (16)

By Postulates 5 and 6

H^=-12⁢m⁢ℏ2⁢d2d⁢x2+V⁢(x) (17)

Now we can take the hamiltonian of some time-independent wave function and by Postulate 5

H^⁢ηn⁢(x)=-12⁢m⁢ℏ2⁢d2⁢ηnd⁢x2+V⁢(x)⁢ηn⁢(x)=En⁢ηn⁢(x) (18)

Equation 18 is the time-independent Schrodinger Equation. We will now derive a time-dependent form of the Schrodinger Equation. From Equation 17 we can substitute a time-dependent wave function ψ⁢(x,t)

H^⁢ψ⁢(x,t)=-12⁢m⁢ℏ2⁢∂2⁡ψ⁢(x,t)∂⁡x2+V⁢(x)⁢ψ⁢(x,t) (19)

and from the Schrodinger Equation (14), we have

i⁢ℏ⁢∂∂⁡t⁢ψ⁢(x,t)=H^⁢ψ⁢(x,t)=-12⁢m⁢ℏ2⁢∂2⁡ψ⁢(x,t)∂⁡x2+V⁢(x)⁢ψ⁢(x,t)i⁢ℏ⁢∂∂⁡t⁢ψ⁢(x,t)=-12⁢m⁢ℏ2⁢∂2⁡ψ⁢(x,t)∂⁡x2+V⁢(x)⁢ψ⁢(x,t) (20)

where the second line is the time-dependent Schrodinger Equation. Importantly, |Ψ⁢(x,t)|2 is a probability density function.

4.3 Compatibility

Two observables A and B are compatible if the measurement of one is certain not to change a measurement of the other. For instance, the eigenstate of a given measurement of B, B^⁢βn=Bn⁢βn for some positive integer n may not necessarily coincide with a previous measurement of A, A^⁢αn=An⁢αn.

4.3.1 Compatibility Theorem

For any two physical observables A and B with corresponding operators A^ and B^, any one of the following statements implies the other two.

  • •

    A and B are compatible observables

  • •

    A^ and B^ possess a common eigenbasis

  • •

    A^ and B^ commute.

Thus since it is clear that the position and momentum operators do not commute, the measurement of one must change a measurement of the other. Which is a conceptual justification for Heisenberg’s uncertainty principle :)

4.4 Heisenberg Uncertainty Principle

The Heisenberg Uncertainty principle states that the uncertainties of A and B for any state Ψt⁢(x) satisfy

Δ⁢A^⋅Δ⁢B^≥12⁢∣c∣ (21)

Where we define c=A^⁢B^-B^⁢A^ to be the commutator of A^ and B^, and we take the uncertainty Δ⁢A^ in the measurement of an observable A to be

Δ⁢A^=⟨A^2⟩-⟨A^⟩2 (22)

We will now prove three lemmas which we will use to derive the Generalized Uncertainty Relation. Afterwards, we will use the Generalized Uncertainty Relation to derive the Heisenberg Uncertainty Principle.

Where ⟨A^⟩ is the expectation value of A^ we define

A^′=A^-⟨A^⟩ and B^′=B^-⟨B^⟩ (23)

Since a Hermitian operator has real eigenvalues, the difference between a Hermitian operator and a scalar also has real eigenvalues. Since A^′ and B^′ are both the difference between a Hermitian operator and a scalar, they both have real eigenvalues. Since the eigenvalues of A^′ and B^′ are real, both of these operators are Hermitian by the derivation in Section 3.2. Now by definition of A^′ and B^′

A^′⁢B′^-B′^⁢A′^=(A^-⟨A^⟩)⁢(B^-⟨B^⟩)-(B^-⟨B^⟩)⁢(A^-⟨A^⟩)A^′⁢B′^-B′^⁢A′^=A^⁢B^-⟨A^⟩⁢B^-A^⁢⟨B^⟩+⟨A^⟩⁢⟨B^⟩-(B^⁢A^-⟨B^⟩⁢A^-B^⁢⟨A^⟩+⟨B^⟩⁢⟨A^⟩) (24)

Since ⟨B^⟩ and ⟨A^⟩ are both scalars, ⟨B^⟩⁢A^=A^⁢⟨B^⟩ and the above equation simplifies to Lemma 2

A^′⁢B′^-B′^⁢A′^=A^⁢B^-B^⁢A^ (25)

Now by the definition of A^′ and since A^′ is hermitian (Lemma 1, 23)

⟨A^′⁢Ψ⁢(x)|A^′⁢Ψ⁢(x)⟩=⟨Ψ⁢(x)|A^′⁣2⁢Ψ⁢(x)⟩=⟨Ψ⁢(x)|[A^-⟨A^⟩]2⁢Ψ⁢(x)⟩⟨A^′⁢Ψ⁢(x)|A^′⁢Ψ⁢(x)⟩=⟨Ψ⁢(x)|[A^2-A^⁢⟨A^⟩-⟨A^⟩⁢A^+⟨A^⟩2]⁢Ψ⁢(x)⟩ (26)

Again since ⟨A^⟩ is a scalar A^⁢⟨A^⟩=⟨A^⟩⁢A^ we have

⟨A^′⁢Ψ⁢(x)|A^′⁢Ψ⁢(x)⟩=⟨Ψ⁢(x)|[A^2-2⁢A^⁢⟨A^⟩+⟨A^⟩2]⁢Ψ⁢(x)⟩ (27)

By the definition of the inner product and the properties of integrals

⟨A^′⁢Ψ⁢(x)|A^′⁢Ψ⁢(x)⟩=⟨Ψ⁢(x)|A^2⁢Ψ⁢(x)⟩-⟨Ψ⁢(x)|2⁢A^⁢⟨A^⟩⁢Ψ⁢(x)⟩+⟨Ψ⁢(x)|⟨A^⟩2⁢Ψ⁢(x)⟩⟨A^′⁢Ψ⁢(x)|A^′⁢Ψ⁢(x)⟩=⟨Ψ⁢(x)|A^2⁢Ψ⁢(x)⟩-2⁢⟨A^⟩⁢⟨Ψ⁢(x)|A^⁢Ψ⁢(x)⟩+⟨A^⟩2⁢⟨Ψ⁢(x)|Ψ⁢(x)⟩ (28)

Assuming the physical system must exist somewhere, Ψ⁢(x) has a norm of 1

⟨A^′⁢Ψ⁢(x)|A^′⁢Ψ⁢(x)⟩=⟨Ψ⁢(x)|A^2⁢Ψ⁢(x)⟩-2⁢⟨A^⟩⁢⟨Ψ⁢(x)|A^⁢Ψ⁢(x)⟩+⟨A^⟩2 (29)

By the definition of expectation value

⟨A^′⁢Ψ⁢(x)|A^′⁢Ψ⁢(x)⟩=⟨A^2⟩-2⁢⟨A^⟩+⟨A^⟩2 (30)

By the definition of uncertainty (Equation 22) we have lemma 3

⟨A^′⁢Ψ⁢(x)|A^′⁢Ψ⁢(x)⟩=(Δ⁢A^)2 (31)

⟨Ψ⁢(x)|[A^⁢B^-B^⁢A^]⁢Ψ⁢(x)⟩=⟨Ψ⁢(x)|[A^′⁢B^′-B^′⁢A^′]⁢Ψ⁢(x)⟩ (32)

⟨Ψ⁢(x)|[A^⁢B^-B^⁢A^]⁢Ψ⁢(x)⟩=⟨Ψ⁢(x)|A^′⁢B^′⁢Ψ⁢(x)⟩-⟨Ψ⁢(x)|B^′⁢A^′⁢Ψ⁢(x)⟩ (33)

⟨Ψ⁢(x)|[A^⁢B^-B^⁢A^]⁢Ψ⁢(x)⟩=⟨A^′⁢Ψ⁢(x)|B^′⁢Ψ⁢(x)⟩-⟨B^′⁢Ψ⁢(x)|A^′⁢Ψ⁢(x)⟩ (34)

For the complex number z=a+b⁢i for real a and b we define Im(z)=b. Thus for z-z* we have z-z*=2⁢Im(z)⁢i and this yields the result

⟨Ψ⁢(x)|[A^⁢B^-B^⁢A^]⁢Ψ⁢(x)⟩=2⁢Im⁢⟨A^′⁢Ψ⁢(x)|B^′⁢Ψ⁢(x)⟩⁢i (36)
